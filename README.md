# 📊 PhonePe User Analytics Dashboard (Tableau)

## 📌 Project Overview

This project is an interactive Tableau dashboard built to analyze PhonePe user transaction data. It helps in understanding transaction patterns, user behavior, and overall performance using visual analytics.

---

## 🎯 Objectives

* Analyze total transactions and total amount
* Understand transaction types distribution
* Visualize state-wise transaction data
* Track transaction trends over time
* Compare success vs failed transactions
* Analyze device usage (Android vs iOS)

---

## 📊 Dashboard Features

* 📌 KPI Cards (Total Transactions & Total Amount)
* 🌍 Map Visualization (Transactions by State)
* 📊 Bar Chart (Transaction Type Distribution)
* 📈 Line Chart (Transaction Trend)
* 🥧 Pie Chart (Device Distribution)
* 🚦 Status Chart (Success vs Failed)

---

## 🛠 Tools & Technologies

* Tableau Desktop / Tableau Public
* CSV Dataset

---

## 📁 Files Included

* `phonepe_dataset_cleaned.csv` → Dataset used
* `phonepe_dashboard.twbx` → Tableau Packaged Workbook
* `dashboard.png` → Dashboard Preview

---

## 🚀 How to Use

1. Download the `.twbx` file
2. Open it using Tableau Desktop or Tableau Public
3. Explore the dashboard and interact with visualizations

---

## 📈 Key Insights

* Most transactions are successful compared to failed ones
* Recharge and Money Transfer are the most frequent transaction types
* Transaction distribution varies across different states
* Majority of users are using Android devices
